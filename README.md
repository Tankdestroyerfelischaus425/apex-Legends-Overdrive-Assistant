## 📌 Overview

This tool assists players in Apex Legends: Overdrive. It provides visual data and mechanical help during matches. The assistant runs on Windows and works within the existing game engine. It tracks player positions and item locations to give you information you might miss during fast games.

## ⚙️ Features

This assistant includes several tools to help your gameplay.

### Aimbot
The aimbot tool calculates target paths. It moves your cursor to track enemies when you hold your fire key. This helps maintain accuracy during long-range fights. You can adjust the speed of the movement in the settings menu.

### ESP (Extra Sensory Perception)
The ESP feature draws boxes around players. This lets you see where enemies are through walls. It displays health bars, team information, and distance markers. These visuals help you plan your moves before you encounter an opponent.

### Ability Prediction (Cyborg Legends)
The assistant tracks the cooldowns and casting times of Cyborg Legends abilities. It notifies you when an enemy can use an ultimate move. This information allows you to time your attacks or retreat to safety.

### Item ESP
This tool highlights loot on the floor. It filters items by rarity. You can set the tool to ignore common gear and only show high-tier weapons or shields. This saves time during loot phases.

## 🛠️ Troubleshooting

If the tool does not work, check these common issues.

*   Disable your antivirus software or add an exclusion for the assistant folder. Some antivirus tools stop the software because it reads game memory.
*   Ensure the game runs in Borderless Windowed mode. This allows the overlays to display on top of the game.
*   Run the assistant as an administrator. Right-click the icon and choose Run as Administrator.
*   Update your GPU drivers through the software provided by NVIDIA or AMD.
*   Disable any other screen recording or overlay software, as these can conflict with the assistant.
